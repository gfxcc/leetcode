1249. Minimum Remove to Make Valid Parentheses.md
```python  
class Solution:  
    def minRemoveToMakeValid(self, s: str) -> str:  
        stack = []  
        remove = set()  
        for i, c in enumerate(s):  
            if c == '(':  
                stack.append(i)  
            elif c == ')':  
                if stack:  
                    stack.pop()  
                else:  
                    remove.add(i)  
        for i in stack:  
            remove.add(i)  
        return ''.join(c for i, c in enumerate(s) if i not in remove)  
```
