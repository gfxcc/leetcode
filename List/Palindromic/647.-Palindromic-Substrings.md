
	O(N^2) - None Manacher's Algorithm  
```python  
class Solution:  
    def countSubstrings(self, s: str) -> int:  
        if not s: return 0  
        rst = len(s)  
        i = 0  
        while i < len(s):  
            j = i  
            while i+1 < len(s) and s[i] == s[i+1]:  
                i += 1  
                rst += i-j  
            rr = i  
            while j - 1 >= 0 and i + 1 < len(s) and s[j-1] == s[i+1]:  
                j -= 1  
                i += 1  
                rst += 1  
            i = rr + 1  
        return rst  
```
